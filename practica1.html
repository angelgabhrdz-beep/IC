“Introducción al Desarrollo Web”

“El desarrollo web es el proceso de crear y mantener sitios web. Combina diferentes habilidades y disciplinas, incluyendo diseño, programación y gestión de contenidos.”

¿Qué es HTML?

: HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Describe la estructura de una página web mediante etiquetas

Las etiquetas HTML definen elementos como encabezados, párrafos, enlaces, imágenes y muchos más.

¿Ques CSS?

CSS (Cascading Style Sheets) es un lenguaje que se utiliza para describir la presentación de un documento HTML. Controla aspectos como colores, fuentes, diseños y responsividad.

CSS permite separar el contenido de su presentación visual, haciendo que el código sea más mantenible.
